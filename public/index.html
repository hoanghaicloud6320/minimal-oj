Nộp bài

Workflow đề xuất

1. Tạo bài

Điền slug, đề Markdown, config, generator test và generator đáp án. Slug là định danh dùng trong URL và thư mục lưu bài.

2. Refresh testcase

Server compile `gentest.cpp` và `gen_answer_from_test.cpp`, sinh lại toàn bộ `.in/.ans` trong thư mục bài.

3. Submit

Học viên dán code C++20. Server compile bằng `g++`, chạy từng test, rồi so sánh token output với đáp án.

Trust model: code được chạy trực tiếp trên máy chủ nội bộ. Dễ dùng trước, sandbox/container để mở rộng sau.
